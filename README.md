# Wii cursor theme/pack
Go to the releases tab and follow these instructions. You can also do this by cloning the repo, but the releases have convenient .tar and .zip files.

## Windows
The .zip file contains the Windows theme. 
### Install
- Unpack it in `C:\Windows\Cursors` so that the .inf file inside is in the path `C:\Windows\Cursors\Wii\install.inf`
- Right-click it and press the `Install` option

> [!NOTE]
> You may need to go to `Settings -> Bluetooth & devices -> Mouse -> Additional mouse settings -> Pointers` and select Wii in the Scheme dropdown.

## Linux (tested on Ubuntu)
The .tar file contains the Linux/GNOME theme.
### Install
- For a user only install, extract the files (the Wii folder) inside `~/.local/share/icons/`
- For a systemwide install, extract the files inside `/usr/share/icons/`
> [!NOTE]
> You may need to get the GNOME Tweaks application and change the mouse/pointer theme to Wii.
